README

FSS — F10 Simple Stack

LEMP stack installer & site manager for Ubuntu 24.04. Fully local, modular, single CLI entry point.

Stack

NGINX 1.30.1 (HTTP/3) PHP 8.4 FPM MariaDB 11.4 Redis (Socket) phpMyAdmin Filebrowser (Fork build: add features compress, permission OS, chown UI) WP-CLI ionCube Fail2Ban iptables (v4+v6) Bad Bot Blocker TCP BBR Let's Encrypt

Get Started

wget https://vps.fio.link/fss-bin.zip
unzip fss-bin.zip -d fss && cd fss
bash fss-install

Add Site

SSL: le (Let's Encrypt) · self (Self-signed) · none (HTTP only)

Mode: --mode multi (default, 1 user per site) · --mode single (1 user, many sites)

WordPress

fss-site add app example.com -php 8.4 -app wordpress -ssl le

General PHP App

fss-site add app example.com -php 8.4 -app general -ssl le

Static Site

fss-site add static example.com -ssl le

Reverse Proxy — Single Backend

fss-site add proxy example.com -backends '127.0.0.1:3000' -ssl le

Reverse Proxy — Load Balancer

fss-site add proxy example.com -backends '127.0.0.1:3000,127.0.0.1:3001' -ssl le

Reverse Proxy — Path-based Routing

fss-site add proxy example.com -routes '/api=127.0.0.1:3000,/app=127.0.0.1:4000' -ssl le

Remove Site

fss-site rm app example.com
fss-site rm example.com

Auto-detects type. Smart removal: keeps user if other domains exist.

Uninstall

fss-uninstall

FSS Paths

  • Binaries: /usr/local/fss/
  • Configs: /opt/fss/conf/
  • Secrets: /opt/fss/data/
  • Logs: /var/log/fss/

Post-Install

  • phpMyAdmin: https://your-ip/pma
  • Filebrowser: https://your-ip/fm
  • MariaDB root: /opt/fss/data/.mysql_root_password
  • Filebrowser admin: /opt/fss/data/.filebrowser_admin
  • Site credentials: /home/<user>/.summary-domain.md

Site Directory

/home/<user>/
├── domain.com/
│   └── public_html/
├── logs/
│   └── domain.com/
│       ├── nginx/
│       └── php/
├── tmp/php_sessions/
└── .summary-domain.com.md
